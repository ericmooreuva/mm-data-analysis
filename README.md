Introduction to Eric's March Madness 2026 Bracket Predictor
This is my web scraper and statistics predictor that I made for my STAT1602's March Madness Bracket 2026. 
I originally had the idea to make some kind of personal project towards this bracket when my professor, Professor Peterson, told the class to join up. 
I also didn't know what exactly to do, until I remembered the term web scraper as I was looking at sports-reference's data. 
I looked up on youtube, as well as an article, on web scraping. From those sources, I was successfully able to gather my data needed for my predictions.
